Apresentação do meu Website

Esta página elenca as capacidades que desenvolvi ao deste projeto, com uma descrição de cada uma das webapps.

    Portfolio:

  • É uma "vitrine digital" do trabalho realizado durante o curso da cadeira de Programação Web. Funciona como um hub central para as todas as páginas do projeto.

    • Página das Bandas:

  • Apresenta uma coleção de álbuns e músicas. Demonstra competência na criação de aplicações web dinâmicas e interativas com Python e Django. Para mim é especialmente significativa na modelação de dados em Django.

    • Página do Curso:

  • Oferece informações abrangentes sobre o meu percurso de Engenharia Informática. Aborda disciplinas, conteúdos programáticos e projetos desenvolvidos.

    • Página do Blog:

  • Procura criar uma comunidade de utilizadores pela partilha de artigos de variados temas. Promove a troca de conhecimentos e experiências entre a toda a comunidade.

    • Página de Meteorologia:

  • Utiliza uma API desenvolvida a partir do IPMA. Fornece informações meteorológicas de maneira acessível e envolvente. Demonstra habilidades na integração de APIs externas e na criação de serviços personalizados.

    • Página de Login:

  • Oferece aos usuários a possibilidade de acederem e interagir com todas as páginas do projeto. Possui recursos de registo local e com uma conta Google. Garante uma experiência de usuário confiável e segura.

Tecnologias Utilizadas:

  • Django
